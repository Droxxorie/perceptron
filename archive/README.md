# Original 2022 sources

This archive preserves three scripts that best represent the original L3 Physics project:

- `rosenblatt_binary_2022.py`: binary 0-vs-1 Rosenblatt perceptron and visualizations.
- `multiclass_network_2022.py`: one-hidden-layer 0–9 classifier and evaluation plots.
- `resize_mnist_2022.py`: nearest-neighbor reduction from 28×28 to 14×14 pixels.

Their contents are intentionally unchanged. They expect CSV and `.npy` files at historical relative paths, execute work at import time, and are not the supported runnable interface. See `src/perceptron_from_scratch/` for the corrected package and `notebooks/perceptron_mnist.ipynb` for the reproducible study.
